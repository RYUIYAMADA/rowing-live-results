📋 登録シート 全項目定義書

Google Drive の master/ フォルダにアップロードする3つのCSVファイルの全項目を定義します。

① ファイル構成

ファイル名 内容 必須 アップロード先
tournament.csv 大会情報（大会名・会場・YouTube URLなど） 必須 master/ フォルダ
schedule.csv レース一覧（種目・日程・ラウンドごとの1行1レース） 必須 master/ フォルダ
entries.csv エントリー情報（レーンごとのクルー名・所属） 必須 master/ フォルダ

② tournament.csv — 大会情報 key / value 形式

1列目に項目名（key）、2列目に値（value）を入力します。順番は問いません。
プロパティ名
（key列に入力）		 日本語の項目名 必須 型・形式 説明・入力例
race_name 大会名 必須 テキスト サイトのタイトルに表示される大会の正式名称
例: 全日本マスターズレガッタ 第100回大会
例: 第42回石川県ボート選手権大会
venue 会場名 任意 テキスト 開催会場の名称。コース距離も含めて記載するとわかりやすい
例: 手取川ダム湖 1000mコース
例: 諏訪湖漕艇場 2000mコース
course_length コース距離（m） 任意 数値（整数） 大会全体のデフォルトコース距離。省略時は 1000 として扱う
レースごとに異なる場合は schedule.csv の course_length 列で個別に上書き可能
例: 500 / 1000 / 2000
dates 開催日（補足） 任意 日付・カンマ区切り 通常は schedule.csv の date 列から自動収集されるため 空欄で構わない
schedule.csv に含まれない日付（予備日・閉会式日など）を追加したい場合のみ記入
例: 2025-09-14,2025-09-15
youtube_url YouTube Live URL 任意 テキスト（URL） YouTube Liveの配信URLを入力するとサイト上に動画が埋め込まれる。配信がない場合は空欄
例: https://www.youtube.com/watch?v=xxxxxxxxxxxxxxx
tournament.csv のサンプル key,value race_name,全日本マスターズレガッタ 第100回大会 venue,手取川ダム湖 1000mコース course_length,1000 dates, ← 空欄でOK（schedule.csvから自動収集） youtube_url, ← 配信がない場合は空欄

③ schedule.csv — レース一覧 1行 = 1レース

1行目はヘッダー行。2行目以降に1レース1行で入力します。レース番号順に並べることを推奨。
プロパティ名
（列ヘッダー）		 日本語の項目名 必須 型・形式 説明・入力例・選択肢
race_no レース番号 必須 数値（整数） 大会内でユニークな番号。1から始まる連番を推奨。CSV計測ファイルのR番号と一致させること
例: 1 / 2 / 3 …
event_name 種目名 必須 テキスト サイト上に表示される種目の正式名称
例: 男子シングルスカル / 女子ダブルスカル / 混成クォドルプル
category 性別区分 必須 テキスト（固定値） 性別を表すコード。フィルタ機能で使用される
M 男子（Male） W 女子（Women） X 混成（Mixed）
age_group 年齢・クラス区分 必須 テキスト（固定値） 選手のクラス・年齢区分を表すコード
G 一般（General・オープン）
J ジュニア（Junior・18歳以下）
U23 アンダー23（23歳以下）
M マスターズ（Masters・27歳以上）
round ラウンド（レース種別） 必須 テキスト（固定値） レースのラウンドを表すコード。フィルタ・並び順に使用される
H 予選（Heat）
R 敗者復活戦（Repechage）
SF 準決勝（Semifinal）
FA 決勝A（Final A・1位決定戦）
FB 決勝B（Final B・7位決定戦）
RK 順位決定戦（Ranking）
date レース実施日 必須 日付（文字列） レースが行われる日付。Excelでは必ず「文字列」書式で入力すること（自動変換に注意）
形式: YYYY-MM-DD（年4桁-月2桁-日2桁）
例: 2025-09-14 / 2025-09-15
time スタート予定時刻 必須 時刻（文字列） レースのスタート予定時刻。Excelでは必ず「文字列」書式で入力すること
形式: HH:MM（24時間表記・時2桁:分2桁）
例: 09:00 / 13:30 / 08:08
event_code 種目コード 任意 テキスト フィルタ機能で同一種目をまとめるためのコード。省略可（省略時は category + "_" + 艇種で自動生成）
例: M_1X（男子シングルスカル）/ W_2X（女子ダブルスカル）/ X_4+（混成フォア）
course_length このレースのコース距離（m） 任意 数値（整数） このレースだけ大会デフォルトと異なる距離の場合に指定。省略時は tournament.csv の course_length を使用
1000m大会に500m種目が混在する場合などに使用
例: 500（500m短距離種目）
schedule.csv のサンプル（2日間大会・500m種目混在の例） race_no,event_name,category,age_group,round,date,time,event_code,course_length 1,男子シングルスカル,M,G,FA,2025-09-14,09:00,M_1X, ← course_length 空欄 = 大会デフォルト(1000m)を使用 2,女子シングルスカル,W,G,FA,2025-09-14,09:08,W_1X, 3,男子ダブルスカル,M,G,FA,2025-09-14,09:16,M_2X, 4,女子ダブルスカル,W,G,FA,2025-09-14,09:24,W_2X, 5,男子クォドルプル,M,G,FA,2025-09-14,09:32,M_4X, 6,女子クォドルプル,W,G,FA,2025-09-14,09:40,W_4X, 7,男子シングルスカル,M,G,H,2025-09-14,10:00,M_1X, 8,女子シングルスカル,W,G,H,2025-09-14,10:08,W_1X, 9,男子ジュニアシングルスカル,M,J,FA,2025-09-15,09:00,M_1X,500 ← 500m種目のみ明示指定 10,女子ジュニアシングルスカル,W,J,FA,2025-09-15,09:08,W_1X,500

④ entries.csv — エントリー情報 1行 = 1クルー

1行目はヘッダー行。2行目以降に1クルー1行で入力します。同じ race_no のクルーはレーン番号順に並べることを推奨。
プロパティ名
（列ヘッダー）		 日本語の項目名 必須 型・形式 説明・入力例
race_no レース番号 必須 数値（整数） このクルーが出場するレース番号。schedule.csv の race_no と必ず一致させること
例: 1 / 5
lane レーン番号 必須 数値（整数） 1から始まる整数。同じ race_no 内でレーン番号は重複不可
例: 1 / 2 / 3 …
crew_name クルー名・選手名 必須 テキスト 選手名（個人）またはクルー名。複数人の場合は / （スペース・スラッシュ・スペース）で区切る
シングル: 山田太郎
ダブル: 高橋雄介 / 小林誠
クォド: 中島拓 / 前田亮 / 野村健 / 大野聡
affiliation 所属団体名 必須 テキスト クルーの所属クラブ・学校・チーム名。フィルタ検索の対象になる
例: 諏訪漕艇クラブ
例: 長野県立松本高等学校
例: 長野漕艇協会
棄権（DNS）について：エントリーに登録されていて当日出漕しなかったクルーは「棄権（DNS）」として自動表示されます。entries.csv からは削除せず、そのまま登録しておいてください。
entries.csv のサンプル race_no,lane,crew_name,affiliation 1,1,山田太郎,諏訪漕艇クラブ 1,2,鈴木次郎,松本ボートクラブ 1,3,佐藤健一,長野漕艇協会 1,4,木村四郎,上田ローイング 2,1,田中花子,岡谷漕艇クラブ 2,2,伊藤明美,飯田ローイング 2,3,渡辺麻衣,諏訪漕艇クラブ 2,4,中村さくら,松本ボートクラブ 3,1,高橋雄介 / 小林誠,長野漕艇協会 ← ダブルスカル：「 / 」で区切る 3,2,加藤大輔 / 吉田拓也,諏訪漕艇クラブ 5,1,中島拓 / 前田亮 / 野村健 / 大野聡,諏訪漕艇クラブ ← クォドルプル：4名を「 / 」で区切る 5,2,後藤慎 / 岡田悠 / 長谷川涼 / 近藤豊,松本ボートクラブ

⑤ Excel・Googleスプレッドシートで作成する場合の注意点

該当列 注意事項
date（レース実施日） Excelが 2025/9/14 などに自動変換してしまうため、セルの書式を「文字列」に設定してから入力する。
すでに変換されてしまった場合は先頭にシングルクォートを付けて '2025-09-14 と入力するか、書式を変更してから再入力する。
time（スタート時刻） Excelが時刻型に自動変換してしまうため、セルの書式を「文字列」に設定してから入力する。
09:00 のように先頭がゼロの場合も正しく保持される。
CSV保存形式 保存時に 「CSV UTF-8（コンマ区切り）」 を選択する。BOM付きでも自動対応済み。
「CSV（コンマ区切り）」（旧形式 Shift-JIS）を選ぶと文字化けする場合があるため注意。
空行・空列 1行目はヘッダー行のみ。データの途中や末尾に空行を入れないこと。
Googleスプレッドシート CSVに変換せず直接スプレッドシートとして master/ フォルダに置いても読み込み可能。
ファイル名を schedule・entries・tournament にすること（拡張子不要）。

⑥ アップロード手順

  1. Google Drive の大会フォルダを開く
  2. master/ フォルダに移動
  3. tournament.csv・schedule.csv・entries.csv の3ファイルをドラッグ＆ドロップ
  4. GASエディタで runImportMaster() を実行（または5分後に自動処理される）
  5. サイトに大会スケジュールとエントリーが反映される ✅
⚠️ 内容を更新したい場合：ファイルを上書きアップロードして再度 runImportMaster() を実行してください。
Driveで「置き換え」を選択すると同じファイルIDで更新されます（新規アップロードでもOK）。
