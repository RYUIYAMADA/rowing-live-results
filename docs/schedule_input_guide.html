📋 スケジュール登録シート 入力ガイド

概要：大会のレーススケジュールとエントリー情報を2枚のCSVファイルに入力し、Google Driveの master/ フォルダにアップロードするだけで自動反映されます。

① ファイル構成

ファイル名 内容 アップロード先
schedule.csv レース一覧（種目・日時・ラウンド） master/ フォルダ
entries.csv エントリー情報（レース番号・レーン・クルー名・所属） master/ フォルダ

② schedule.csv の書き方

カラム説明

カラム名 必須 説明 入力例
race_no 必須 レース番号（1始まりの整数） 1, 2, 3…
event_name 必須 種目名（表示に使用） 男子シングルスカル
category 必須 性別区分 M=男子 / W=女子 / X=混成
age_group 必須 年齢区分 G=一般 / J=ジュニア / U23 / M=マスターズ
round 必須 ラウンド FA=決勝A / FB=決勝B / SF=準決勝 / H=予選 / R=敗者復活 / RK=順位決定
date 必須 レース日（YYYY-MM-DD形式） 2025-09-14
time 必須 スタート時刻（HH:MM形式） 09:00
event_code 任意 種目コード（フィルタに使用。省略時は自動生成） M_1X / W_2X / X_4+
course_length 任意 このレースのコース距離（m）。省略時は大会デフォルト距離を使用。1000m大会の中に500m種目がある場合に指定 500 / 1000 / 2000（空欄可）

③ entries.csv の書き方

カラム説明

カラム名 必須 説明 入力例
race_no 必須 schedule.csv の race_no と一致させる 1
lane 必須 レーン番号（1始まり） 1, 2, 3…
crew_name 必須 クルー名（艇名・選手名） 山田太郎
affiliation 必須 所属団体名 諏訪漕艇クラブ
複数人クルーの場合：crew_name に「 / 」（スラッシュ前後にスペース）で区切って入力してください。
例: 山田太郎 / 鈴木次郎（ダブル）　田中A / 佐藤B / 鈴木C / 木村D（クォド）

④ Excelで作成する場合の注意点

項目 注意事項
日付列（date） セルの書式を「文字列」に設定してから入力してください。Excelが自動変換してしまう場合は '2025-09-14（先頭にシングルクォート）
時刻列（time） 同様に文字列として入力してください。09:00 の形式で入力します。
保存形式 「CSV UTF-8（コンマ区切り）」で保存してください（BOM付きでも自動対応済み）
空行・空列 1行目はヘッダー行のみ。データの途中に空行を入れないでください。

⑤ アップロード手順

  1. Google Drive の大会フォルダを開く
  2. master/ フォルダに移動
  3. schedule.csv と entries.csv をドラッグ＆ドロップ
  4. GASエディタで runImportMaster() を実行（または5分後に自動処理）
  5. サイトに大会スケジュールが反映されます ✅
⚠️ 更新の場合：ファイルを上書きアップロードして再度 runImportMaster() を実行してください。 Driveで「置き換え」を選択すると同じファイルIDで更新されます。
