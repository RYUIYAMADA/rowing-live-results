📋 スケジュール登録シート 入力ガイド

概要：大会のレーススケジュールとエントリー情報を2枚のCSVファイルに入力し、Google Driveの master/ フォルダにアップロードするだけで自動反映されます。

① ファイル構成

ファイル名 内容 アップロード先
tournament.csv 大会情報（大会名・会場・開催日・YouTube URLなど） master/ フォルダ
schedule.csv レース一覧（種目・日時・ラウンド） master/ フォルダ
entries.csv エントリー情報（レース番号・レーン・クルー名・所属） master/ フォルダ

② tournament.csv の書き方（大会情報）

大会名・会場・YouTube URLなど、大会全体の情報を登録します。
key（項目名）と value（値）の2列構成です。

登録できる項目

key（項目名） 日本語の意味 必須 型 入力例
race_name 大会名 必須 テキスト 全日本マスターズレガッタ 第100回大会
venue 会場名 任意 テキスト 手取川ダム湖 1000mコース
course_length コース距離（m） 任意 数値（整数） 1000　（500・1000・2000 のいずれか。省略時は1000）
dates 開催日（追加指定） 任意 日付・カンマ区切り 2025-09-14,2025-09-15
（通常は schedule.csv の date 列から自動収集されるため空欄可）
youtube_url YouTube Live URL 任意 テキスト（URL） https://www.youtube.com/watch?v=xxxxxxx
（配信がない場合は空欄）

サンプルデータ（tournament.csv）

key,value race_name,全日本マスターズレガッタ 第100回大会 venue,手取川ダム湖 1000mコース course_length,1000 dates, youtube_url,

③ schedule.csv の書き方（レース一覧）

カラム説明

カラム名 日本語の意味 必須 型 入力例
race_no レース番号 必須 数値（整数） 1, 2, 3…　（1から始まる連番）
event_name 種目名 必須 テキスト 男子シングルスカル　／　女子クォドルプル
category 性別区分 必須 テキスト（固定値） M = 男子　／　W = 女子　／　X = 混成
age_group 年齢・クラス区分 必須 テキスト（固定値） G = 一般　／　J = ジュニア　／　U23 = U23　／　M = マスターズ
round ラウンド（レースの種別） 必須 テキスト（固定値） FA = 決勝A　／　FB = 決勝B　／　SF = 準決勝
H = 予選　／　R = 敗者復活　／　RK = 順位決定
date レース実施日 必須 日付（文字列） 2025-09-14　（YYYY-MM-DD形式。Excelでは文字列書式で入力）
time スタート時刻 必須 時刻（文字列） 09:00　（HH:MM形式・24時間表記。Excelでは文字列書式で入力）
event_code 種目コード（フィルタ用） 任意 テキスト M_1X　／　W_2X　／　X_4+　（省略可）
course_length このレースのコース距離（m） 任意 数値（整数） 500　（1000m大会の中に500m種目がある場合のみ指定。省略時は tournament.csv の course_length を使用）

サンプルデータ（schedule.csv）

④ entries.csv の書き方（エントリー情報）

カラム説明

カラム名 日本語の意味 必須 型 入力例
race_no レース番号 必須 数値（整数） 1　（schedule.csv の race_no と一致させる）
lane レーン番号 必須 数値（整数） 1, 2, 3…　（1から始まる整数）
crew_name クルー名・選手名 必須 テキスト 山田太郎　／　高橋雄介 / 小林誠（複数人は「 / 」で区切る）
affiliation 所属団体名 必須 テキスト 諏訪漕艇クラブ　／　長野県立松本高等学校
複数人クルーの場合：crew_name に「 / 」（スラッシュ前後にスペース）で区切って入力してください。
例: 山田太郎 / 鈴木次郎（ダブル）　田中A / 佐藤B / 鈴木C / 木村D（クォド）

⑤ Excelで作成する場合の注意点

項目 注意事項
日付列（date） セルの書式を「文字列」に設定してから入力してください。Excelが自動変換してしまう場合は '2025-09-14（先頭にシングルクォート）
時刻列（time） 同様に文字列として入力してください。09:00 の形式で入力します。
保存形式 「CSV UTF-8（コンマ区切り）」で保存してください（BOM付きでも自動対応済み）
空行・空列 1行目はヘッダー行のみ。データの途中に空行を入れないでください。

⑥ アップロード手順

  1. Google Drive の大会フォルダを開く
  2. master/ フォルダに移動
  3. tournament.csv・schedule.csv・entries.csv をドラッグ＆ドロップ
  4. GASエディタで runImportMaster() を実行（または5分後に自動処理）
  5. サイトに大会スケジュールが反映されます ✅
⚠️ 更新の場合：ファイルを上書きアップロードして再度 runImportMaster() を実行してください。 Driveで「置き換え」を選択すると同じファイルIDで更新されます。
